# lfe-cli-hello-world

## Description
This is a POC project to demonstrate lfe a cross between erlang and lisp.

## Tech stack
- lfe

## Docker stack
- lfex/opensuse

## To run
`sudo ./install.sh -u`

## To stop (optional)
`sudo ./install.sh -d`

## To see help
`sudo ./install.sh -h`
